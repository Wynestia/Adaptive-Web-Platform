# Adaptive-Web-Platform
Web application ที่ผู้ใช้สามารถ "สั่ง" เปลี่ยนแปลงระบบได้ด้วยภาษาธรรมชาติ (natural language) โดยมี LLM เป็นแกนกลางที่แปลคำสั่งเป็นการแก้ไขจริง — ทั้ง UI, feature, และ workflow — แบบ dynamic  แนวคิดหลัก: LLM มีอิสระในการตัดสินใจและดำเนินการเอง แต่ทุกการเปลี่ยนแปลง มองเห็นได้ และ ย้อนกลับได้ ผ่านระบบ admin ที่กำกับดูแลและตรวจสอบได้เสมอ
